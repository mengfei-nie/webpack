#项目初始化安装
***
##1、安装所需要的包
`npm install`

##2、运行指令
`npm run dev`

##3、预览效果
输入网址：<br>
   http://localhost:8188/
   
效果如下：<br>
   ![Image text](https://github.com/duanliang920/webpack-website/blob/master/dev.jpg)

##4、编译打包
`npm run build`
   
效果如下：<br>
   ![Image text](https://github.com/duanliang920/webpack-website/blob/master/build.gif)

温馨提示：如果没有安装node和webpack的朋友，请先安装。
